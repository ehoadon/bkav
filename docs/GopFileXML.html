Gộp & Chuẩn Hóa File XML/TXT

Ghi chú chức năng:

  • Hỗ trợ chọn nhiều file XML/TXT và gộp thành 1 hoặc nhiều file.
  • Tự động loại bỏ khai báo XML <?xml ...?> và dọn khoảng trắng thừa.
  • Sắp xếp theo InvoiceDate, có thể lọc chọn ngày cụ thể.
  • Chia batch nếu >100 file.
  • Chọn xuất .xml hoặc .txt.
